AGENTPULSE FRONTEND INSTALLATION

1. The templates/ and static/ folders are already placed in the correct project root.
2. The authentication redirect settings were added to config/settings.py.
3. Run:
   python manage.py check
   python manage.py runserver
4. Open http://127.0.0.1:8000/
5. Sign in using a Django user. Create one if needed:
   python manage.py createsuperuser
